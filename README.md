# RegardeMoiCa
RegardeMoiCa is a screenshot, image capture, and vector annotation application. It is built for modern Gnome Desktop Environment. It is build on the Adwaita libraries. 


## What the Name ??? 
RegardeMoiCa is from French 'Regarde Moi Ca' which translate to 'Look at that!'
